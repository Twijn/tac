tac.extensions.updater v1.3.0

Provides auto-update functionality for TAC libraries and (eventually) the TAC system itself via lib/updater.lua. This extension checks for updates on startup and provides commands to check for and install updates.

UpdaterExtension.init(tac)

View source

Initialize the updater extension This function is called when the extension is loaded by TAC. It performs the following actions: 1. Checks for available updates on startup and notifies the user if any are found 2. Registers the 'updater' command with 'check' and 'update' subcommands The extension will silently check for updates and only print a message if updates are available. If no updates are found, no message is displayed.

Parameters:
  • tac (table): The TAC instance that provides command registration and hooks
